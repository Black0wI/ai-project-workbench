# Codex Workspace

This directory contains repository-local Codex assets.

## Skills

Skills are reusable workflows for recurring tasks. Each skill lives in `.codex/skills/<skill-name>/SKILL.md`.

Included skills:

- `project-kickoff`: turn a rough idea into a scoped project setup.
- `spec-writer`: create or improve implementation specs.
- `architecture-review`: review architecture, boundaries, and risks.
- `implementation-review`: review code changes before merge.

## Guidance

- Keep skills short and operational.
- Prefer checklists and concrete outputs.
- Update skills when repeated project patterns emerge.
- Do not store secrets or private customer data here.
