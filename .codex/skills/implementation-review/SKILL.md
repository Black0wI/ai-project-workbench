# Implementation Review

Use this skill for review mode.

## Workflow

1. Inspect changed files.
2. Compare implementation against the active spec.
3. Prioritize bugs, regressions, missing tests, and security issues.
4. Keep findings specific and actionable.
5. If no issues are found, state remaining risks or test gaps.

## Review Focus

- Correctness
- Edge cases
- Security and privacy
- AI prompt/tool safety
- Observability
- Test coverage
- Deployment compatibility
