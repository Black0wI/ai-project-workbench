# Example Obsidian Vault Structure

Copy this structure into a private Obsidian vault outside the public repository.

## Folders

- `00-inbox`: quick capture
- `10-project`: project context and maps
- `20-specs`: spec drafts before promotion to `docs/specs`
- `30-decisions`: decision drafts before ADR promotion
- `40-prompts`: prompt drafts before promotion to `prompts`
- `50-research`: research notes and references
- `90-archive`: old notes

## Rule

Do not store secrets, tokens, customer data, or private production details in a vault that may be exported to this repository.
