# Cost Review Checklist

Use this before approving AWS infrastructure or always-on services.

- [ ] Cost profile is documented.
- [ ] Always-on resources are justified.
- [ ] Pay-per-use options were considered.
- [ ] Dev/staging/prod cost differences are considered.
- [ ] Budget alerts are planned.
- [ ] Required cost tags are included.
- [ ] Storage growth is estimated.
- [ ] Data transfer costs are considered.
- [ ] Cost review is linked from the PR when infrastructure is touched.
