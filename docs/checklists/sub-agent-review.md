# Sub-Agent Review Checklist

Use this before accepting delegated work.

- [ ] Delegated goal was clear.
- [ ] File/module ownership was respected.
- [ ] No unrelated files were changed.
- [ ] No user work was reverted.
- [ ] No secrets or private data were exposed.
- [ ] AWS, Cloudflare, GitHub, database, or release decisions were not implemented without validation.
- [ ] Output satisfies the assigned scope.
- [ ] Main agent reviewed the result.
- [ ] Required tests/checks were run or the gap is documented.
- [ ] Integration risks are understood.
