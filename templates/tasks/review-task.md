# Review Task

## Review Target

- Branch:
- PR:
- Spec:

## Review Focus

- Correctness
- Security and privacy
- Tests/checks
- AI behavior
- AWS cost and service fit
- Cloudflare DNS safety
- PostgreSQL usage for relational data
- DESIGN.md compliance for UI

## Output Format

Findings first, ordered by severity.

For each finding:

- File and line
- Severity
- Problem
- Impact
- Suggested fix

## Residual Risk

List test gaps or operational risks even if no blocking issues are found.
