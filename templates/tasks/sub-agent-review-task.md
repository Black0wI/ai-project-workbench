# Sub-Agent Review Task

## Goal

Review a specific area for bugs, regressions, security, or spec compliance.

## Review Target

- Branch/commit:
- Files/modules:
- Spec:

## Focus Areas

- Correctness
- Edge cases
- Security and privacy
- AI behavior and tool safety
- DESIGN.md compliance, if UI
- AWS/Cloudflare/GitHub/PostgreSQL/IaC governance, if touched
- Tests/checks

## Constraints

- Read-only unless explicitly asked to patch.
- Findings first, ordered by severity.
- Cite exact files and lines.
- Do not invent issues.
- If no issues are found, say so and list residual risk.

## Expected Final Response

- Findings
- Open questions
- Test gaps
- Residual risk
